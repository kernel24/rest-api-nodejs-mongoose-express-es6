## Restful API with Mongoose & Express & React & mocha

This is simple CRUD for user preferences using Node.js, MongoDB and React.

## Requirements
* Node.js v8.2.1
* MongoDB v3.4.6

## Prerequisite

* pure ES6/7 and Node.js 8.x
* React/InfernoJS
* Express
* MongoDB
* Async/Await
* Session handling
* Test with mocha and chai

###### ... install
```
$ npm install
$ npm start
```
